Licensing

In the following the licenses for the TouchGFX Grapical Framework are described. This section covers the following parts:

  • The core framework as precompiled libraries, TouchGFX driver and Core files for the following targets:
    • Windows/gcc
    • Windows/MSVS
    • Linux/gcc
    • ARM Cortex-M3/IAR
    • ARM Cortex-M4F/IAR
  • TouchGFX examples, templates and demos
  • Tools for converting images, texts and fonts
  • The manual you are reading now
